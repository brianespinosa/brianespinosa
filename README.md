### ⚙️ GitHub Actions

A small family of composable actions I reach for on every Next.js repo. All share the same [`release-action`](https://github.com/brianespinosa/release-action) for automated semver releases.

- **[checkout-setup-node-install](https://github.com/brianespinosa/checkout-setup-node-install)** · Bundles checkout, Node/yarn setup, and install into one step.
- **[job-root-cache](https://github.com/brianespinosa/job-root-cache)** · Save, restore, and cleanup a run-scoped workspace cache across parallel jobs.
- **[next-build-cache](https://github.com/brianespinosa/next-build-cache)** · Caches the Next.js build output for faster CI.
- **[release-action](https://github.com/brianespinosa/release-action)** · Reusable workflow for semver releases, changelogs, and alias tag management.
